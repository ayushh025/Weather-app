# 🌦️ Weather App

A modern and responsive weather application built using **HTML, CSS, and JavaScript**.
It provides real-time weather updates along with hourly forecasts using OpenWeather API.

🔗 https://your-live-link-here.com

---

## ✨ Features

* 🌍 Search weather by **city name**
* 📍 Get weather using **current location (Geolocation API)**
* 🌡️ Shows **current temperature**
* ☁️ Displays **weather condition (Cloudy, Rain, Sunny, etc.)**
* 🕒 **Hourly forecast** (next 24 hours)
* 📱 Fully **responsive design**
* ⚠️ Error handling for invalid city names
* ⚡ Fast and smooth UI experience

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* OpenWeather API
